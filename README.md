## Getting start

    1. install
        /front, /back : npm i
    2. dotenv
        /back => .env.example
    3. initiallize db
        /back : sequelize db:create && sequelize db:seed
    4. run
        webpack-dev-server: /front npm run build
        app: /back npm run dev
